#CNN Feed Processor - Interview Submission

This project will translate the CNN story feed to the desired format of: 

```js

[
 {
    url: '...',      
    headline: '...',
    imageUrl: '...',
    byLine: '...'
 }
]

```

## Getting Started

- Clone Repository
- **npm install** 

## Running the Project

```
npm start
```
