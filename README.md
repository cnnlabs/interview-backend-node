#CNN Feed Processor - Interview Submission

This project will translate the CNN story feed to the desired format of: 

```js

[
 {
    url: '...',      
    headline: '...',
    imageUrl: '...',
    byLine: '...'
 }
]

```

## Getting Started

- Clone Repository
- **npm install** 
- **npm start**

## Testing the Project

To test the project and run all unit tests and eslint do the following:

```
npm test
```
